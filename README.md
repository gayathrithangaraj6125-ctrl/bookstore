### 📚 AI-Powered Bookstore Web Application (Django) ###
📌 Overview

This is a full-stack Bookstore Web Application developed using Django as part of my major academic project. The system allows users to browse, search, and manage books with secure authentication and role-based access control.

The project includes separate functionalities for normal users and admin users, with an integrated admin dashboard for managing the system efficiently.

### 🚀 Features ###
## 👤 User Features
User registration and login system
Browse and search books
View book details
Secure session management
## 🛠️ Admin Features
Admin dashboard for full control
Add, update, and delete books
Manage registered users
Role-based access control (staff/superuser)
## 🧠 Key Functionalities
Authentication & Authorization system
Role-based dashboard visibility
CRUD operations for books
Responsive UI design
Secure backend with Django
## 🛠️ Tech Stack
Backend: Python, Django
Frontend: HTML, CSS, Bootstrap
Database: SQLite / MySQL
Tools: VS Code, Git, GitHub
